## Editors

### Strudel REPL
https://strudel.cc

### Strudel REPL embedded
https://strudel.cc/technical-manual/project-start/#embedding-the-strudel-repl
Disadvantages: stable version only

### Flok
https://flok.cc
Advantages: collaborate
Disadvantages: less Strudel-specific tools (like reference or suggestions)

### VS code plugin
https://marketplace.visualstudio.com/items?itemName=cmillsdev.strudelvs
Advantages: save code, play in editor
Disadvantages: no support for sliders, limited support for visuals

### Hydra
https://hydra.ojack.xyz
Advantages: explore examples, randomise parameters
Disadvantages: no support for Strudel
